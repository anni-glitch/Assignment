# Malware Research Dashboard

A sleek, controlled-environment dashboard for distributing and analyzing cybersecurity cases such as Viruses, Ransomware, and RATs.

## Features
- 🚀 Glassmorphism UI with hover effects
- 🌌 Interactive particle background
- ⚡ Vanilla Tilt implementation for 3D cards
- 🔗 Direct download links for analysis files

## Deployment Status
This project is statically built using HTML, CSS, and JS. It is fully ready to be deployed on **GitHub Pages**, Vercel, or Netlify without any build step.
